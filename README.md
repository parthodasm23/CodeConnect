# CodeConnect – A Modern Full-Stack Interview Platform

CodeConnect is a full-stack, real-time coding interview platform designed to make technical interviews smoother, faster, and more interactive.

It brings together code execution, video calling, chatting, screen sharing, and interview collaboration—everything in one place.

---

## What This Project Includes

- **Live Coding Environment** powered by a VSCode-style editor
- **1-on-1 Video Interview Rooms** with camera & mic controls
- **Real-time Chat Messaging** inside interview rooms
- **Code Execution** with automatic validation against test cases
- **Success & Failure Feedback**, including confetti celebrations
- **Practice Mode** for solving problems individually
- **Room Locking** to ensure only two people join an interview
- **Inngest Background Jobs** for async workflows
- **TanStack Query** for smooth data fetching
- **Clerk Authentication** for secure login
- **Production-ready Git & GitHub workflow**
